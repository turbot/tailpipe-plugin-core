# Core Plugin for Tailpipe

[Tailpipe](https://tailpipe.io) is an open-source CLI tool that allows you to collect logs and query them with SQL.


The Core plugin for Tailpipe provides the `file` source which can be for any tabler which supports artifact sources.    

The Core plugin is automatically installed and updated by Tailpipe - there is no need to manually install it 

- **[Get started →](https://hub.tailpipe.io/plugins/turbot/aws)**
- Community: [Join #tailpipe on Slack →](https://turbot.com/community/join)
